## 🎯 推荐方案：Railway（最简单）

Railway 提供 $5/月的免费额度，支持持久化存储，非常适合这个项目。

### 部署步骤

#### 1. 注册 Railway 账号
1. 访问 https://railway.app
2. 使用 GitHub 账号登录（推荐）或邮箱注册
3. **需要绑定信用卡**（但会获得 $5 免费额度）

#### 2. 部署项目

**方法A：直接从 GitHub 部署（推荐）**

1. 将项目上传到 GitHub：
   ```bash
   cd 蜀来驰二手车网站
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/slc-used-car.git
   git push -u origin main
   ```

2. 在 Railway 中创建项目：
   - 登录 Railway 控制台
   - 点击 "New Project" → "Deploy from GitHub repo"
   - 选择刚才创建的仓库
   - Railway 会自动检测为 Python 项目

3. 配置环境变量：
   - 在 Railway 项目设置中添加：
   ```
   PORT = 8000
   FLASK_ENV = production
   ```

4. 等待部署完成（约2-3分钟）

**方法B：通过 Railway CLI 部署**

```bash
# 1. 安装 Railway CLI
npm install -g @railway/cli

# 2. 登录
railway login

# 3. 初始化项目
cd 蜀来驰二手车网站
railway init

# 4. 部署
railway up

# 5. 查看状态
railway status
```

#### 3. 获取访问地址

部署成功后，Railway 会提供类似 `https://slc-used-car.up.railway.app` 的访问地址。

## 🆓 备选方案：Deta（完全免费）

Deta 提供完全免费的云部署，不需要信用卡。

### 部署步骤

#### 1. 安装 Deta CLI

```bash
# macOS / Linux
curl -fsSL https://get.deta.dev/space-cli.sh | sh

# Windows (PowerShell)
iwr https://get.deta.dev/space-cli.ps1 -useb | iex
```

#### 2. 登录 Deta

```bash
space login
```

#### 3. 部署项目

```bash
cd 蜀来驰二手车网站

# 创建新项目
space new

# 推送到云端
space push
```

#### 4. 获取访问地址

```bash
space dev  # 本地开发
space deploy  # 部署到云端
```

访问地址：`https://your-project.deta.dev`

**注意**：Deta 免费层限制：
- 每月 100GB 带宽
- 2GB 存储空间
- 每天 10万次请求

## ❓ 常见问题

### Q: 部署后数据会丢失吗？

**Railway** 和 **Deta** 都支持持久化存储，数据不会丢失。

**Render** 免费层不支持持久化，每次重新部署数据会重置。

### Q: 访问速度慢怎么办？

- 国内用户建议使用 Railway（节点在新加坡/日本）
- 或使用阿里云/腾讯云（需要付费）

### Q: 如何备份数据？

定期导出数据库文件 `shulaichi.db`：
```bash
# Railway 中下载
railway run sqlite3 shulaichi.db ".backup shulaichi.db"
```
