# 🚗 蜀来驰二手车网站 - 快速部署到云端

## ⚡ 最简方案：使用 Railway 部署（推荐）

Railway 提供 $5/月免费额度，支持持久化存储，部署简单。

### 第一步：准备项目

1. 将项目上传到 **GitHub**（必需）

```bash
# 在蜀来驰二手车网站目录下执行
cd 蜀来驰二手车网站

# 初始化 Git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 创建 GitHub 仓库后，添加远程仓库
git remote add origin https://github.com/你的用户名/蜀来驰二手车.git
git branch -M main
git push -u origin main
```

### 第二步：部署到 Railway

1. 访问 **https://railway.app**
2. 点击 **"Login"** → 使用 **GitHub 登录**（推荐）
3. 绑定信用卡（会获得 $5 免费额度，不需要立即充值）
4. 点击 **"New Project"**
5. 选择 **"Deploy from GitHub repo"**
6. 选择刚才创建的仓库
7. Railway 会自动检测并部署

### 第三步：配置环境变量

1. 在 Railway 项目中，点击 **"Variables"**
2. 添加：
   ```
   PORT = 8000
   FLASK_ENV = production
   ```

### 第四步：等待部署完成

- 约 2-3 分钟
- 看到绿色 ✅ 表示部署成功

### 第五步：获取访问地址

Railway 会提供类似：`https://slc-used-car.up.railway.app`

## 🆓 完全免费方案：Deta Cloud

如果不想绑定信用卡，使用 Deta（完全免费）。

### 安装 Deta CLI

```bash
# macOS / Linux
curl -fsSL https://get.deta.dev/space-cli.sh | sh

# Windows (PowerShell)
iwr https://get.deta.dev/space-cli.ps1 -useb | iex
```

### 登录并部署

```bash
# 登录
space login

# 进入项目目录
cd 蜀来驰二手车网站

# 创建项目
space new

# 部署
space deploy
```

获取地址：`https://your-project.deta.dev`
