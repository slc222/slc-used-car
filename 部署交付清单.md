## 🚀 推荐部署方式

### Railway（最简单）

1. 注册 https://railway.app（GitHub登录）
2. 创建新项目 → Deploy from GitHub
3. 上传本项目到GitHub，选择仓库
4. 自动部署完成

### Deta（完全免费）

1. 安装：`curl -fsSL https://get.deta.dev/space-cli.sh | sh`
2. 登录：`space login`
3. 部署：`cd 蜀来驰二手车网站 && space deploy`
