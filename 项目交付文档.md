# 蜀来驰二手车网站 - 项目交付文档

## 项目概述

为**蜀来驰二手车**开发的二手车展示网站，包含前台展示和后台管理系统。

### 基本信息
- **车商名称**：蜀来驰二手车
- **地址**：四川省威远县
- **展厅面积**：500平方
- **主营**：10万以内短公里数二手车
- **抖音粉丝**：2.2万

## 技术实现

### 技术栈
- **后端框架**：Python Flask 2.3+
- **数据库**：SQLite（轻量、易用）
- **前端技术**：HTML5 + CSS3 + Vanilla JavaScript
- **UI框架**：Font Awesome 6.4 图标库
- **部署方式**：Gunicorn + Nginx

### 项目规模
- **文件数量**：21个
- **代码行数**：约6000行
- **模板文件**：9个HTML
- **样式文件**：1个CSS

## 快速开始

### 本地运行

```bash
# 1. 进入项目目录
cd 蜀来驰二手车网站

# 2. 安装依赖
pip install flask flask-sqlalchemy werkzeug

# 3. 启动网站
python app.py

# 4. 访问
# 前台：http://localhost:5000
# 后台：http://localhost:5000/admin/login
```

详见 `DEPLOY.md` 部署指南文档

## 数据库说明

使用SQLite数据库，文件名为 `shulaichi.db`

### 数据表
1. **Admin** - 管理员账户
2. **Car** - 车辆信息
3. **Slider** - 轮播图
4. **SiteSettings** - 网站设置
5. **VisitLog** - 访问日志

## 注意事项

⚠️ **安全提醒**：
1. 请在部署后立即修改默认密码
2. 建议启用HTTPS加密传输
3. 定期备份数据库文件

⚠️ **文件存储**：
- 上传的图片存储在 `static/uploads/` 目录
- 建议定期清理不需要的图片

⚠️ **数据备份**：
```bash
# 备份数据库
cp shulaichi.db shulaichi.db.backup
```
